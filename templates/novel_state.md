# 🕒 当前连载状态追踪 (Novel State)

这是高频读写文件。每次执行完 `/继续连载`，本文件都会被 `/更新记忆` 脚本自动覆写。
它记录了主角**在当前章节**的瞬时状态，防止出现“上一章查克拉耗尽，这一章满血放大”的逻辑硬伤。

## 1. 实时定位 (Current Position)
- **当前卷/章**：卷一 / 第 1 章
- **当前时间点**：木叶X年X月（距离大事件【如九尾之乱】还有 X 天）
- **当前地理位置**：木叶村 / 死亡森林 / 铁之国边境

## 2. 主角实时状态 (Protagonist Status)
- **查克拉剩余量**：100% (满状态) / 50% (警戒线) / 10% (枯竭边缘)
- **肉体战损程度**：无伤 / 轻伤 / 重伤 (影响结印速度与体术发挥)
- **底牌/大招冷却池**：
  - [忍术A]：可用
  - [外挂B]：冷却中（还需 3 章恢复）

## 3. 情报与短期目标 (Intel & Short-Term Goal)
- **主角目前已知的情报**：（例如：已经知道带土没死，但不知道长门的具体位置）
- **主角不知道的情报 (读者视角上帝信息)**：（反派正在暗中策划针对主角的陷阱）
- **未来 1-3 章的短期目标**：（例如：活过今晚的暗杀 / 凑齐购买起爆符的赏金）

## 4. 压力余波与可见反应 (Emotional Continuity)
- **本章压力来源**：（角色实际经历了什么事件，不直接写“他很绝望”）
- **可见反应**：（身体动作、物件处理、环境注意点、对话潜台词）
- **行动选择**：（角色在压力下做了什么）
- **下章承接点**：（下一章需要延续的行为惯性或未解决的小冲突）
- **情绪呈现提醒**：不要用抽象词反复重说同一情绪；用事件、动作、物件和对话潜台词承载心理。

## 5. 重点关注配角状态 (Key Companions Status)
- [配角A]：目前位于 [地点]，状态 [安全/濒死]。
- [配角B]：目前位于 [地点]，对主角的信任度 [提升/下降]。
